Cambridge GIS Data - CDD
========================

This is the Cambridge GIS data for the Community Development Department category only.  For the complete Cambridge GIS dataset see the [cambridgegis_data](https://github.com/cambridgegis/cambridgegis_data) repository.  The [README](https://github.com/cambridgegis/cambridgegis_data/blob/master/README.md) in that repository contains more details about the Cambridge GIS open data.
